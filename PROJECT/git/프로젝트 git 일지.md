# 프로젝트 git 일지

## 5/11(월)

- 오른쪽 위 `Fork` 로 내 `repository` 에 포크하기
- 원하는 위치에 `디렉토리` 만들기
- 해당 위치에서 우클릭 후 `Git bash here`
- `git init`
- 확인하기 : `git config --global user.name & user email`

- 초대받은 `repository` 를 `clone or douwnload`로 복사
- `git remote add origin` + `복사한 url`
- 가능한 동작 확인하기 : `git remote -v`



## 5/12(화)
